# Agent Configuration

## Agent skills

### Issue tracker

GitHub Issues via `gh` CLI. PRs are not a request surface for triage. See `docs/agents/issue-tracker.md`.

### Triage labels

Default label names: `needs-triage`, `needs-info`, `ready-for-agent`, `ready-for-human`, `wontfix`. See `docs/agents/triage-labels.md`.

### Domain docs

Single-context layout. `CONTEXT.md` at root, ADRs in `docs/adr/`. See `docs/agents/domain.md`.
