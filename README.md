# Theobase

Information system for the Seventh-day Adventist Church — collecting data from the grassroots level to the world church hierarchy.

## Status

Early development — Phase 0 (Bootstrap) in progress.

## Stack

- **Backend**: Hono on Cloudflare Workers
- **Frontend**: Vite + React + TanStack Router + TanStack Query (PWA)
- **Language**: TypeScript

## Development

```bash
npm install
npm run dev
```
